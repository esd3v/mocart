## Mocart
An example of psd-to-html project based on **Mocart e-Commerce Website PSD Template** by **PSD Freebies** https://psdfreebies.com/psd/mocart-e-commerce-website-psd-template/

## System Requirements
- Node.js 15+

## Run development server
```
  npm run dev
```
## Build project
```bash
  npm run build
```
